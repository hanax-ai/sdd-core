# Operational Registers

Living, committed registers — the high-churn ASSIGNMENT side of standards
defined in governance-framework. Each register names the framework standard it
implements (one-way dependency, constitution Article III) and never redefines
it. Current registers:

- [`deliverables.md`](deliverables.md) — implements framework standard ST-001
  (Deliverables & Ownership): actual owners, status, review dates, exceptions.

(The Runbook & Cadence Register lives in `../knowledge/instructions.md` §1;
evidence policy in `../records/`.)
