# Coordination

One claim file per active workstream under `claims/<agent-or-workstream>.md`,
containing: owner (stable name), scope (files + topic), start date, expected
completion. Check ALL existing claims before claiming; overlapping intent means STOP
and coordinate — never parallel work on a claimed scope. Close/remove the claim after
integration. Stale claims: 14 days idle → lead pings; +7 days silent → lead may close
as abandoned. See ../../COLLABORATION.md.
