# Contributions

One uniquely named file per contributor or workstream: `<agent-or-workstream>.md`
(humans: GitHub handle; agents: `claude-<role>-<assigned-name-or-short-id>`, e.g.
claude-research-alpha — bare roles are not valid identifiers). Write only in your own
file; never
edit another contributor's file — respond in yours. The synthesis lead integrates
accepted contributions into the canonical item README. Competing proposals stay
separate until reconciled. See ../../COLLABORATION.md.
